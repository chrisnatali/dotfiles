# my dotfiles

Link them to home via:
```
ln -sb ~/src/dotfiles/.bashrc ~
ln -sb ~/src/dotfiles/.bash_aliases ~
.
.
.
```

For directories, do:
```
cp -rs ~/src/dotfiles/.xmonad ~/
```

## Requirements

- xbindkeys
- xdotool
- xmonad
  - dmenu
  - dzen2
- urxvt-font-size (perl module for resizing terminal font)
